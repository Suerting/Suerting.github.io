# 事件修饰符

<div id="root">

     <!-- 阻止事件冒泡 ，点击按钮，冒泡到div元素,由内到外-->

# 滚轮

scroll是页面滚动条滚动会触发的事件，而wheel是鼠标滚轮滚动会触发的事件。 一旦滚动条到底部后，滑动鼠标滚轮继续滚动，wheel就会一直触发，而scroll不会触发。 scroll需等待回调函数执行完毕才触发，wheel无需等待回调函数执行完毕就可以触发

<div id="root">
    <ul @scroll="scroll" class="list1">
                <Li>1</Li>
                <Li>2</Li>
                <Li>3</Li>
                <Li>4</Li>
    </ul>
    <!-- 绑定滚动事件 @wheel滚动轮的滚动-->
    <ul @wheel.passive="wheel" class="list2">
                <Li>1</Li>
                <Li>2</Li>
                <Li>3</Li>
                <Li>4</Li>
    </ul>
<div id="root">

<script type="text/javascript">
     Vue.config.productionTip=false // 阻止vue在启动时生成生产提示
     const vm=new Vue({
          el:'#root',
          data:({
                
            }),
            methods:{
                // 事件的回调都配置在methods中，最终会在vm上
                //函数都写成普通函数，this表示vue实例，不要写成箭头函数，箭头函数中的this是window
                scroll(){
                    console.log('scroll滚动')
                },
                wheel(){
                    console.log('wheel')
                    for(let i=0;i<10000;i++){
                        console.log ('djfh')
                    }
                    console.log('累死')
                }
            }
     })
</script>

<div id="root">

            },
             computed:{

                }
            },
          })

<div id="root">
    <h2>今天天气{{info}}，是的加{{a}}</h2>
    第二种方法，回调函数
    <button @click="clickweather">切换天气</button>
</div>

<script type="text/javascript">
        Vue.config.productionTip=false;
        const vm=new Vue ({
            el:'#root',
            data:{
                name:'vue6监听属性案例',
                weather:true,
                a:0
            },
            methods: {
                clickweather(){
                    this.weather=!this.weather,
                    this.a++
                }
            },
            computed:{
                info(){
                    return this.weather ? '炎热':'凉快'
                }
            },
            // 监视（在控制台）
            //监视对象内部值的改变（一层）
            watch:{
                // 监视谁
                weather:{
                    immediate:true,//初始化时让handle调用一下
                    //当weather发生改变时调用handle
                    handler(newValue,oldValue){
                        console.log('weather被修改，handler被调用',newValue,oldValue)
                    }
                },
                info:{
                    immediate:true,//初始化时让handle调用一下
                    //当weather发生改变时调用handle
                    handler(newValue,oldValue){
                        console.log('info被修改，handler被调用',newValue,oldValue)
                        console.log('调用前的值',newValue)
                        console.log('调用后的值',oldValue)
                    }
                }
           }
})

也可通过vm进行监视
vm.$watch('weather',{
     immediate:true,//初始化时让handle调用一下
     //当weather发生改变时调用handle
     handler(newValue,oldValue){
         console.log('通过vm实现监视')
         console.log('weather被修改，handler被调用',newValue,oldValue)
         console.log('调用前的值',newValue)
         console.log('调用后的值',oldValue)
      }
})

                // 监视多级结构中某个属性的变化
                'numbers.b':{
                    handler(){
                        console.log('numbers中b的值被修改')
                    }
                },
                // 监视多级属性中所有属性的变化

                    deep:true,//可以检测到对象内部值改变（多层）

                }
            }
})

3.生命周期函数的名字不可更改，但函数的具体内容是程序员根据需求编写的。

7.关于销毁Vue实例

​ 1.销毁后借助Vue开发者工具看不到任何信息。

​ 2.销毁后自定义事件会失效，但原生DOM事件依然有效。

​ 3.一般不会再beforeDestroy操作数据，因为即便操作数据，也不会再触发更新流程了。

     // 页面呈现未经过VUE编译的dom结构
     console.log('beforeMount')

# 28，组件

Vue中使用组件的三大步骤:

​ 一、定义组件(创建组件)

​ 二、注册组件

​ 三、使用组件(写组件标签)

​ 如何定义一个组件?

​ 使用Vue . extend(options )创建，其中options和new Vue(options )时传入的那个options几乎一样， 但也有点区别;

​ 区别如下F:

​ 1.el不要写，为什么?

​ 最终所有的组件都要经过个vm的管理， 由vm中的el决定服务哪个容器。

​ 2.data必须写成函数，为什么? 避免组件被复用时，数据存在引用关系。

​ 备注:使用template 可以配置组件结构。

​ 如何注册组件?

​ 1.局部注册:靠new Vue的时候传入components选项

​ 2.全局注册:靠Vue. component( '组件名' ,组件)

​    	(2).new Vue()配置中: 
    		data函数、methods中的函数、watch中的函 数、computed中的函数它们的this均是[Vue实例对象]。
